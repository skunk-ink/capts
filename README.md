### How to run
Before operating you will need to add your Coinbase API keys to the `keyfile` located under `data/keyfile.dat`. Kraken has not been implemented.

```
python3 CAPTS.py
```

### Dependencies
```
pip install pandas
pip install numpy
pip install plotly
pip install getch
pip install cbpro
```
